Sobre este Ambiente TechFleet

Este é um ambiente de produção simulado da TechFleet rodando em um cluster Kubernetes local. O objetivo é validar a aplicação web app-portal antes de migrá-la para a nuvem da empresa.

Características da Implantação:

  • Deployment com 3 réplicas (escalável para 5)
  • Service do tipo NodePort (porta 30080)
  • Alta disponibilidade e recuperação automática
  • Customização de conteúdo via ConfigMap

Este sistema demonstra:

  • Escalabilidade - capacidade de aumentar réplicas sob demanda
  • Resiliência - recuperação automática após falhas
  • Balanceamento de carga - distribuição do tráfego entre réplicas
